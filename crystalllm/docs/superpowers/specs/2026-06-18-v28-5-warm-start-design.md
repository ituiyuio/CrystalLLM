# v28.5 Warm-Start 扩规模设计

> **Q: v25 (24L) warm-start + 4 层扩展到 28L, 4000 步训练, PPL 能从 2.44 改进多少?**
> **A: 目标 PPL < 2.2 (小幅改进). 主要价值: 验证 warm-start 路线可行.**

## 1. 背景

### 1.1 v28 失败原因

- 数据分布不匹配 (ext_v23 短文 vs val 长文)
- 训练步数不够 (2000 vs v25 4000)
- 无 warm-start

### 1.2 v28.5 策略

**保守扩展**: 只扩展模型, 不改变数据
- 数据: v24 train (19K) - 与 v25 一致
- 模型: v25 (24L) + 4L 扩展 = 28L
- Warm-start: 复用 v25 24L 权重
- 训练: 4000 步, LR=2e-5 (小, 用于微调)

## 2. 架构

| 配置 | 值 |
|---|---|
| Layers | 28 (24 warm + 4 new) |
| embd | 1280 (同 v25) |
| head | 20 (同 v25) |
| T | 512 |
| D_Z | 256 |
| 参数量 | ~580M (vs v25 476M, 1.22x) |

## 3. 训练

| 参数 | 值 | 备注 |
|---|---|---|
| B | 4 (同 v25) | |
| T | 512 | |
| LR | 2e-5 | 微调, 比 v25 小 |
| STEPS | 4000 | 同 v25 |
| Warmup | 400 | 10% |
| Data | v24 train (19K) | 避免分布偏移 |

## 4. 评估标准

| 指标 | 目标 | 最低 |
|---|---|---|
| PPL | < 2.2 | < 2.5 |
| 速度 | < 600ms | < 800ms |
| 训练时间 | < 30min | < 1h |

## 5. 风险

1. **Warm-start 不兼容**: 维度必须匹配 (✓ 24L 直接复用)
2. **新 4L 学不到**: 随机初始化, LR 小可能不动 → 必要时提高 LR
3. **速度退化**: 多 4 层 = 多 17% 时间 → 可接受
